# dcard-crawler
## Introduction
- crawler for [Dcard](https://www.dcard.tw/)
- using Python3

## Install

```sh
git clone https://github.com/noname0930/dcard-crawler.git
cd dcard-crawler
pip install -r requirements.txt
```

## Run
- Normal

    ```sh
    python bot.py
    ```
    
- with debug message

    ```sh
    python bot.py -- degbug
    ```
